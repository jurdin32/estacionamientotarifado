📱 FLUJO DEL USUARIO
═══════════════════════════════════════════════════════════════════════════

1. ABRE NotificacionScreen
   │
   ├─ Ve indicador en AppBar
   │  ├─ VERDE: Impresora conectada ✅
   │  └─ GRIS: Impresora no conectada ❌
   │
   └─ Ve botón flotante
      ├─ VERDE: Impresora conectada ✅
      └─ GRIS: Impresora no conectada ❌

2. CONFIGURA IMPRESORA (opcional)
   │
   ├─ Presiona indicador en AppBar O botón flotante
   ├─ Se abre pantalla de configuración
   ├─ Escanea impresoras disponibles
   ├─ Selecciona su impresora
   └─ Se conecta automáticamente

3. COMPLETA EL FORMULARIO
   │
   ├─ Placa: ABC-1234 ✓
   ├─ Multa: Selecciona tipo ✓
   ├─ Comprobante: 12345678 ✓
   ├─ Fecha: Hoy a las 14:30 ✓
   ├─ Ubicación: Calle Principal 123 ✓
   ├─ Observación: Vehículo estacionado... ✓ (mín 10 caracteres)
   └─ Fotos: Selecciona 3 evidencias ✓

4. VE BOTÓN "IMPRIMIR TICKET"
   │
   ├─ Aparece: Solo si impresora está CONECTADA
   ├─ Color: Azul
   └─ Ubicación: Encima del botón "Guardar"

5. PRESIONA "IMPRIMIR TICKET"
   │
   ├─ Abre diálogo de confirmación
   │  └─ Muestra resumen: Placa, tipo, valor, ubicación
   ├─ Selecciona "Imprimir"
   ├─ La impresora imprime el ticket
   └─ Recibe notificación de éxito

6. PRESIONA "GUARDAR MULTA"
   │
   ├─ Se guarda en la base de datos
   ├─ Se suben las 3 fotos al servidor
   ├─ Muestra diálogo de éxito
   └─ Se limpia el formulario

🔒 VALIDACIONES
═══════════════════════════════════════════════════════════════════════════

El botón "IMPRIMIR TICKET" aparece SOLO SI:

  ✓ _gestorImpresora.estaConectada == true
    └─ Es decir, hay una impresora Bluetooth conectada

  Y

  ✓ formularioCompleto == true
    ├─ Placa válida (ABC-1234 o AB-123C)
    ├─ Multa seleccionada
    ├─ Comprobante: 2-8 dígitos
    ├─ Fecha: no puede ser futura
    ├─ Ubicación: mínimo 3 caracteres
    ├─ Observación: mínimo 10 caracteres
    └─ Fotos: exactamente 3
