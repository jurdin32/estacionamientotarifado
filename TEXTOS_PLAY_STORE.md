## Descripción completa

```
SIMERT es el sistema municipal de estacionamiento tarifado del cantón El Guabo. Permite al personal autorizado gestionar en tiempo real el control de estacionamientos, registro de tarjetas, consulta de infracciones y notificaciones de tránsito.

Funcionalidades principales:
• Control de tarjetas de estacionamiento con cuenta regresiva
• Registro y consulta de infracciones de tránsito
• Notificaciones push en tiempo real sobre estacionamientos próximos a vencer
• Impresión de comprobantes vía Bluetooth (impresora térmica)
• Captura de evidencias fotográficas
• Consulta de datos de vehículos por placa
• Sincronización automática con el servidor municipal
• Panel de métricas diarias y mensuales
• Roles y permisos personalizados por usuario
• Beneficios para adultos mayores y personas con discapacidad
```
